# HOUSE-PRICE-PREDICTION-ANALYSIS
House Price Prediction using Python and Machine Learning. This project analyzes housing data, performs data cleaning, exploratory data analysis (EDA), and predicts house prices using Linear Regression.
# House Price Prediction using Python

## Project Overview
This project predicts house prices using Machine Learning. The dataset is analyzed, cleaned, visualized, and used to build a Linear Regression model for price prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Understanding
4. Data Cleaning
5. Exploratory Data Analysis (EDA)
6. Feature Selection
7. Train-Test Split
8. Linear Regression Model
9. Prediction
10. Model Evaluation

## Dataset
- Housing Dataset (CSV)

## Machine Learning Algorithm
- Linear Regression
